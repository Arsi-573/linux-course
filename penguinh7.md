# h7 Maalisuora

#### A) Hei maailma kolmella eri kielellä.

Ajattelin käyttää tässä oppitunnilla läpikäytyjä kieliä, eli Bash, Python ja C. 

#### Bash 
Aloitin luomalla tiedoston "Halojata Maailma" microlla komennolla ´´´micro halojata´´´. Microon kirjoitin syötteeksi:
```
Bash
#!/bin/bash

# Tervehdys
echo "Halojata Muailma!"
```

<img width="342" height="131" alt="image" src="https://github.com/user-attachments/assets/d66a340f-a840-460c-99f5-d80aa7e70a5f" />

Tallensin tekeleeni Microssa ```Ctrl + S``` ja suljin sen ```Ctrl + Q```. Seuraavaksi kokeilin komentoa nykyisessä hakemistossa ```./halojata```, joka kertoi, että oikeuteni eivät tähän leikkiin riitä. Korjasin tilanteen komennolla ```chmod ugo+x /home/juho/halojata```, joka antaa tarvittavat oikeudet, että saan tulosteen annettuani komennon nykyisessä hakemistossa. Tämän jälkeen uudestaan ```./halojata``` ja sain vastaukseksi microssa aikaisemmin kirjoitetun "Halojata Mualima!".
